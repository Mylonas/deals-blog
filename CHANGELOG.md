# Changelog

All notable changes to DealsHub will be documented here.

## [1.0.0] — 2026-06-28

### Added
- Initial public launch of DealsHub Cyprus
- Next.js 15 + Tailwind CSS static site, deployed on Cloudflare Pages
- Markdown-powered blog post system (`/posts/[slug]`)
- First post: Cheapest Fried Chicken per piece in Nicosia (Foody, Wolt, Bolt comparison)
- GitHub Actions CI/CD pipeline for automatic deploys to Cloudflare Pages on push to `master`
- Environment variable support (`NEXT_PUBLIC_SITE_NAME`, `NEXT_PUBLIC_SITE_URL`)
